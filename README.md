This is a demonstration on how to use 'Google Map' in Ionic Framework.<br />

<b>Version used:</b> 2<br /><br />
<b>Plugins used:</b><br />
ionic plugin add cordova-plugin-geolocation<br /><br />

<b>NPM Install:</b><br />
npm install --save @ionic-native/geolocation<br /><br />
